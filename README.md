# GSM Prime Distribuidora

Landing page estática para a GSM Prime Distribuidora, empresa do Rio de Janeiro focada em embalagens, filme stretch, bobinas, fitas e materiais para operações B2B.

## Estrutura

- `index.html`: página principal
- `styles.css`: identidade visual e responsividade
- `script.js`: links de WhatsApp e formulário de cotação
- `assets/`: logo e imagem hero
- `BRAND.md`: identidade inicial, narrativa e dados provisórios

## Dados provisórios

Os números de WhatsApp e e-mails dos fundadores ainda são placeholders e devem ser substituídos pelos contatos reais antes de tráfego pago.
